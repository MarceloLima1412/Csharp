Routes

Methods Route Description
GET api/sensor/{id} Esta rota é disponibilizada para ir buscar todos os dados de um sensor especifico. Nesta rota tem de indicar o id do sensor
GET api/sensor Esta rota é disponibilizada para ir buscar todos os dados de todos os sensores presentes na base de dados
POST api/sensor/insert Esta rota é disponibilizada para inserir dados de um sensor na base de dados
DELETE api/sensor/delete/{id} Esta rota é disponiblizada para eliminar dados de um determinado sensor. Esta rota pode ser utilizada por exemplo para eliminar dados de um sensor avariado
PUT api/sensor/update/{id}/{timestamp} Esta rota é disponibilizada para mudar dados de um sensor num determinado tempo. Esta rota pode ser utilizada por exemplo para atualizar dados de um sensor que num determinado tempo enviou dados errados
GET api/alert Esta rota é disponibilizada para ir buscar todos os dados de todos os alertas presentes na base de dados
POST api/alert/insert Esta rota é disponibilizada para inserir dados de um alerta na base de dados
